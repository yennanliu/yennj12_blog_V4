## 🧠 聲音設計的科學原理

### 為什麼深海/太空音效有療癒效果？

**神經科學基礎**：

```yaml
大腦對聲音的反應:

1. 低頻聲音（20-250 Hz）
   效果:
     • 活化副交感神經系統
     • 降低心率和血壓
     • 誘導深度放鬆狀態

   深海音效:
     • 低頻隆隆聲（100-200 Hz）
     • 模擬深海水壓和洋流

   太空音效:
     • 低頻 Drone（30-150 Hz）
     • 模擬宇宙背景輻射轉換成聲音

2. 粉紅噪音（Pink Noise）
   頻譜: 隨頻率增加，能量遞減（1/f）

   自然界的粉紅噪音:
     • 海浪聲
     • 降雨聲
     • 瀑布聲

   科學證據:
     • 2017 年《Frontiers in Human Neuroscience》研究:
       粉紅噪音可提升深度睡眠品質 75%
     • 2012 年《Journal of Theoretical Biology》:
       粉紅噪音可改善記憶鞏固

3. 無預測性模式
   定義: 聲音有結構但不重複

   為什麼有效:
     • 大腦無法預測下一秒的聲音
     • 避免「習慣化」（Habituation）
     • 持續提供「新奇但不干擾」的刺激

   如何實現:
     • 使用多層次音頻（5-8 層）
     • 每層獨立循環（不同長度）
     • 數學上要數百小時才會重複

4. 缺少高頻刺激
   避免: >4000 Hz 的尖銳聲音

   原因:
     • 高頻 = 警報聲（火災、尖叫）
     • 大腦演化上對高頻高度警覺
     • 無法真正放鬆

   深海/太空的優勢:
     • 自然就缺乏高頻
     • 深海: 水吸收高頻
     • 太空: 真空無介質
```

**心理學效應**：

```
視聽整合（Audiovisual Integration）

視覺: 深邃的海洋
聽覺: 低頻隆隆聲
  ↓
大腦整合
  ↓
強化「深度」、「寧靜」、「安全」的感知
  ↓
更強的放鬆效果（相比單獨音頻 +40%）

研究支持:
  • 2020 年《Nature》研究顯示:
    搭配視覺的環境音，壓力降低效果提升 35%
  • 多感官整合可增強記憶留存（觀眾更容易回訪）
```

### 方法 B：音效層疊法（Sound Layering）

**概念**：將多個音頻層混合，創造豐富的聲景

**優勢**：
- 完全掌控每個元素
- 可調整各層音量和 EQ
- 創造獨特且不會重複的聲音

**音頻層結構（深海範例）**：

```yaml
第 1 層: 低頻 Drone（基底）
  來源:
    • Suno AI 生成純 Drone
    • 或使用 Synthesizer（Serum, Vital）

  頻率範圍: 40-150 Hz
  音量: -12 dB
  效果: Reverb（長混響，模擬水下擴散）

第 2 層: 水流聲
  來源:
    • Freesound.org 下載
    • 或 AI 生成 "gentle water currents"

  頻率範圍: 200-2000 Hz
  音量: -18 dB
  效果:
    • Low-pass filter at 3000Hz
    • Subtle chorus（創造「流動感」）

第 3 層: 氣泡聲
  來源:
    • AI 生成 "underwater bubbles rising"
    • 或採樣真實氣泡聲

  頻率範圍: 500-4000 Hz
  音量: -24 dB（非常細微）
  效果: Random panning（左右隨機移動）

第 4 層: 生物音效
  來源:
    • AI 生成 "distant whale songs, very far"
    • 或授權音效庫

  頻率範圍: 100-1000 Hz
  音量: -20 dB
  觸發: 每 3-5 分鐘一次（隨機）
  效果: Heavy reverb, delay

第 5 層: 環境細節
  來源:
    • 潛艇金屬聲（可選）
    • 遠處船隻（可選）
    • AI 生成

  頻率範圍: 全頻
  音量: -30 dB（幾乎聽不見，但增加深度）
  觸發: 隨機，稀疏

總輸出:
  • Master volume: -3 dB（預留 headroom）
  • EQ: 低切 <30Hz，高切 >8000Hz
  • Compression: 輕微（ratio 2:1）
  • Limiter: -0.5 dB ceiling
```

**使用 Audacity 實作**：

```markdown
Step 1: 準備音頻層

1. 使用 Suno AI 分別生成 5 種音效:
   - "deep low frequency drone"
   - "gentle water current sounds"
   - "underwater bubbles"
   - "distant whale calls"
   - "ambient underwater details"

2. 下載並放入對應資料夾

Step 2: 在 Audacity 中疊加

1. File → Open → 選擇第 1 層（Drone）
2. File → Open → 繼續打開其他層
   （會自動疊加為多音軌）
3. 調整每層音量:
   - 選擇音軌
   - Effect → Amplify
   - 輸入目標 dB（參考上面數值）

Step 3: EQ 處理（重要！）

每一層都應該處理 EQ，避免頻率衝突:

第 1 層（Drone）:
  Effect → Filter Curve EQ
  • High-pass: 40 Hz（移除極低頻）
  • Low-pass: 300 Hz（只保留低頻）

第 2 層（水流）:
  • High-pass: 200 Hz
  • Low-pass: 3000 Hz
  • Boost: 800-1200 Hz +2dB（增加存在感）

第 3 層（氣泡）:
  • High-pass: 500 Hz
  • Low-pass: 6000 Hz

第 4 層（生物）:
  • High-pass: 100 Hz
  • Low-pass: 2000 Hz

Step 4: 空間效果

為每層添加混響:

選擇音軌 → Effect → Reverb
設定（深海）:
  • Room Size: 100%（巨大空間）
  • Damping: 50%（模擬水的吸收）
  • Wet Gain: -10 dB
  • Dry Gain: -3 dB

Step 5: 創造隨機性

對於第 4、5 層（偶發音效）:
  1. 剪輯成小段（5-10 秒）
  2. 手動放置在時間軸的隨機位置
  3. 確保間隔 2-5 分鐘
  4. 每次出現的音量略微不同

Step 6: Master 處理

1. 選擇所有音軌
2. Tracks → Mix and Render
3. 對混音後的音軌:
   Effect → Compressor
     • Threshold: -20 dB
     • Ratio: 2:1
     • Attack: 10 ms
     • Release: 100 ms

4. Effect → Limiter
   • Type: Hard Limit
   • Limit to: -0.5 dB

5. 最終檢查:
   • 播放並聆聽全頻
   • 檢查是否有爆音
   • 確認整體音量一致

Step 7: 導出

File → Export → Export Audio
設定:
  • Format: WAV (Microsoft)（無損）
  • Encoding: Signed 16-bit PCM
  或
  • Format: MP3
  • Quality: 320 kbps（最高品質）

命名: ocean_layered_01_60min.mp3
```
