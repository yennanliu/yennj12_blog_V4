## 🎨 Synthwave 視覺美學的核心DNA

### 與深海/太空視覺的關鍵差異

```yaml
深海/太空視覺:
  色彩: 單一色系（藍、黑為主）
  光線: 柔和、自然、擴散
  動態: 極緩慢（飄移、呼吸）
  氛圍: 平靜、深邃、神秘
  情緒: 放鬆、沉思、療癒
  目標: 讓觀眾「沉浸」並忘記視覺存在

Synthwave/賽博龐克視覺:
  色彩: 強烈對比（霓虹粉、電藍、紫、橙）
  光線: 人造、尖銳、霓虹燈
  動態: 中速（脈衝、掃描、移動）
  氛圍: 都市、科技、復古未來
  情緒: 激勵、懷舊、酷炫
  目標: 讓觀眾感到「能量」和「專注動力」

關鍵挑戰:
  ┌─────────────────────────────────────┐
  │ Synthwave 視覺必須「吸睛」但「不刺眼」│
  │                                       │
  │ 矛盾:                                 │
  │ • 霓虹燈很炫 → 但長時間會眼睛疲勞   │
  │ • 高對比很酷 → 但會分散注意力       │
  │ • 動態很炫 → 但會消耗觀眾精力       │
  │                                       │
  │ 解決方案:                             │
  │ • 降低飽和度 20-30%                  │
  │ • 使用深色背景（護眼）               │
  │ • 動態元素控制在 20-30% 畫面         │
  │ • 週期性變化而非隨機（可預測）       │
  └─────────────────────────────────────┘
```

## 🌆 80 年代視覺元素圖鑑

### 經典元素清單

**1. 日落漸層（Sunset Gradient）**

```yaml
定義: Synthwave 最標誌性的視覺元素

經典配色:
  • 頂部: 深紫 (#1A0B2E)
  • 中上: 紫紅 (#7B2869)
  • 中間: 粉紅 (#E91E63)
  • 中下: 橙紅 (#FF5722)
  • 底部: 深橙 (#FF6F00)

  漸層方向: 垂直（由上至下）

變化版本:
  • 冷色調: 紫 → 藍 → 青
  • 暖色調: 紅 → 橙 → 黃
  • 賽博龐克: 紫 → 粉 → 電藍

使用場景:
  • 背景主色
  • 天空部分
  • 光暈效果

Midjourney 關鍵字:
  「sunset gradient background」
  「pink and purple gradient sky」
  「80s sunset colors」
  「vaporwave gradient」
```

**2. 網格地平線（Grid Horizon）**

```yaml
定義: 象徵數位世界的網格線

視覺特徵:
  • 透視網格線（消失點在中央）
  • 通常在畫面下半部
  • 霓虹色線條（粉紅或藍綠）
  • 地平線處有發光效果

經典比例:
  • 網格佔畫面下方 40-50%
  • 線條間距: 遠處密集，近處稀疏（透視）
  • 線寬: 2-3 px（細線）

光效:
  • 地平線處強烈 bloom（光暈）
  • 網格線半透明（30-50% opacity）
  • 交點處輕微發光

Midjourney 提示詞:
  「perspective grid floor」
  「neon grid lines vanishing point」
  「80s computer grid」
  「tron-style grid」
  「retrowave grid landscape」
```

**3. 霓虹燈（Neon Lights）**

```yaml
物理特性（真實霓虹燈）:
  • 發光氣體管
  • 顏色受氣體種類影響
  • 有自然的光暈（bloom）
  • 邊緣模糊

數位重現要點:
  • 核心顏色: 高飽和度
  • 內發光（Inner Glow）
  • 外發光（Outer Glow）
  • 輕微模糊（Gaussian Blur 2-3px）

經典霓虹色:
  粉紅霓虹: #FF10F0
  電藍霓虹: #00E5FF
  紫色霓虹: #B026FF
  青綠霓虹: #00FFC8
  橙色霓虹: #FF6B35

應用:
  • 文字效果
  • 線條裝飾
  • Logo 邊框
  • 建築物輪廓

Midjourney 提示詞:
  「neon lights glow effect」
  「pink neon signs」
  「bright neon cyan highlights」
  「glowing neon edges」
```

**4. 棕櫚樹剪影（Palm Tree Silhouette）**

```yaml
象徵意義:
  • 80 年代美國西岸文化
  • Miami Vice 影集
  • 加州夢想
  • 熱帶與科技的混合

視覺特徵:
  • 純黑剪影（100% 黑）
  • 對比強烈的背景（日落漸層）
  • 通常在畫面兩側
  • 數量: 3-7 棵

構圖:
  左側: 2-3 棵（高低錯落）
  右側: 2-3 棵
  中央: 保持空曠（主體空間）

Midjourney 提示詞:
  「palm tree silhouettes」
  「black palm trees against sunset」
  「tropical palm trees backlit」
  「miami vice palm trees」
```

**5. 復古跑車（Retro Sports Car）**

```yaml
經典車款:
  • Lamborghini Countach（最標誌性）
  • Ferrari Testarossa
  • DeLorean DMC-12（回到未來）
  • Porsche 911 Turbo

視覺處理:
  • 低角度（仰視）
  • 強烈輪廓光（Rim Light）
  • 霓虹色反光
  • 速度感（motion blur，可選）

位置:
  選項 A: 前景（近距離特寫）
  選項 B: 中景（行駛在網格上）
  選項 C: 剪影（遠景日落前）

Midjourney 提示詞:
  「lamborghini countach neon lights」
  「80s sports car retrowave」
  「delorean cyberpunk style」
  「retro supercar at sunset」
```

**6. 賽博龐克城市（Cyberpunk City）**

```yaml
視覺層次:
  背景層: 巨大建築物剪影
  中景層: 詳細建築（窗戶、霓虹招牌）
  前景層: 街道、車輛

光源:
  • 窗戶: 暖黃光
  • 霓虹招牌: 粉紅、藍、紫
  • 街燈: 橙色
  • 探照燈: 強烈光束（可選）

天氣效果:
  • 輕霧（增加深度感）
  • 細雨（反光效果）
  • 無雲（清晰天空展示漸層）

Midjourney 提示詞:
  「cyberpunk city skyline」
  「neon city at night」
  「blade runner cityscape」
  「futuristic 80s city」
  「towering skyscrapers neon lights」
```

**7. 數位元素（Digital Elements）**

```yaml
類型:
  • 掃描線（Scanlines）
  • 像素化（Pixelation）
  • 故障效果（Glitch）
  • 數據流（Data Stream）
  • HUD 介面（Heads-Up Display）

掃描線:
  • 水平細線（1-2px）
  • 黑色或深色
  • 半透明（10-20% opacity）
  • 覆蓋整個畫面

數據流:
  • Matrix 風格數字/字元
  • 由上至下流動
  • 霓虹綠或電藍
  • 速度: 緩慢到中速

HUD 介面:
  • 四角框線
  • 虛構數據顯示
  • 目標鎖定圖示
  • 科技感文字

使用原則:
  ⚠️ 不要過度使用（會太雜亂）
  ✓ 選擇 1-2 種元素
  ✓ 保持微妙（不搶主視覺）
```

### 方法 B：After Effects（專業級）

```markdown
適合進階用戶，完全掌控

專案架構:

comp_main（主合成，16:9, 1920x1080, 30fps）
├── layer_background（背景圖層）
│   └── midjourney_image.png
├── layer_grid（網格動畫）
│   └── grid_animation
├── layer_scanlines（掃描線）
│   └── scanline_overlay
├── layer_particles（粒子效果）
│   └── neon_particles
├── layer_glow（光暈效果）
│   └── bloom_glow
└── layer_text（文字資訊，可選）
    └── channel_name

────────────────────────────────────

動畫 1: 網格線滾動

1. 創建 Solid Layer（純色圖層）
   Layer → New → Solid
   顏色: 黑色
   命名: Grid Base

2. 應用 Grid 效果
   Effect → Generate → Grid
   設定:
     Width: 1920, Height: 1080
     Width From: Left Edge
     Height From: Top Edge
     Border: 40（網格線寬度）
     Color: #00E5FF（電藍）
     Blending Mode: Invert
     Anchor: 960, 540

3. 添加動畫
   在 Grid 效果的 Anchor 屬性上按 Alt + Click
   輸入表達式:
     ```javascript
     x = 960;
     y = 540 + time * 50; // 每秒向下移動 50 px
     [x, y % 1080]; // 循環回到頂部
     ```

4. 添加透視變形
   Effect → Distort → CC Lens
   或
   手動 3D 變形（更真實）

5. 調整透明度
   Opacity: 30%

────────────────────────────────────

動畫 2: 掃描線效果

1. 創建 Solid Layer
   顏色: 黑色
   命名: Scanlines

2. 應用 Venetian Blinds 效果
   Effect → Transition → Venetian Blinds
   設定:
     Transition Completion: 50%
     Width: 2（細掃描線）
     Direction: 0°（水平）

3. 調整
   Blend Mode: Multiply
   Opacity: 15-20%

────────────────────────────────────

動畫 3: 霓虹光暈

1. 複製背景圖層
   Duplicate Layer

2. 應用 Glow 效果
   Effect → Stylize → Glow
   設定:
     Glow Threshold: 70%
     Glow Radius: 30
     Glow Intensity: 1.0

3. 應用 Gaussian Blur
   Effect → Blur & Sharpen → Gaussian Blur
   Blurriness: 10

4. 調整混合模式
   Blend Mode: Add（相加）
   Opacity: 50%

5. 動畫化（脈衝效果）
   在 Glow Intensity 屬性上按 Alt + Click
   表達式:
     ```javascript
     baseline = 0.8;
     amplitude = 0.3;
     frequency = 0.5; // 每 2 秒一次脈衝
     baseline + Math.sin(time * frequency * Math.PI * 2) * amplitude;
     ```

────────────────────────────────────

動畫 4: 粒子效果（可選）

1. 新增 Solid Layer
   顏色: 黑色
   命名: Particles

2. 應用 CC Particle World
   Effect → Simulation → CC Particle World
   設定:
     Birth Rate: 2（稀疏）
     Longevity: 5 sec
     Velocity: 0.3
     Gravity: 0
     Particle Type: Star
     Birth Color: #00E5FF
     Death Color: #FF10F0
